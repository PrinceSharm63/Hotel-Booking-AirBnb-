Hotel Booking (AirBnb)

Hotel Booking (AirBnb) is a simple and user-friendly hotel booking application where users can explore hotels, view details, and book rooms easily. The project is inspired by AirBnb and focuses on providing a clean, smooth and modern booking experience.

Features :

Browse and explore multiple hotels

View hotel images, price, location and details

Search hotels by location

Simple and clean user interface

Smooth navigation and booking experience

Responsive design (works on all devices)

Purpose of the Project :

The main goal of this project is to create a functional hotel booking system with a modern UI similar to AirBnb.
It helps users understand hotel browsing and booking flow.
